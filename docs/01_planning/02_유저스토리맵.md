## 2. 일일 루틴 플로우

### 2.1 아침 플로우 (07:00~09:00)
```
[앱 알림: "베니가 오늘 하루를 기다리고 있어요 🌱"]
    ↓
[앱 실행 → 메인 홈]
    ↓
[아침 감정 체크인]
 - 감정 선택 (기쁨/평온/피곤/슬픔/불안)
 - 강도 설정 (1~5)
 - 선택: 한 줄 메모
    ↓
[베니 아침 인사 반응]
 - 피곤/슬픔: "오늘도 여기 있을게. 천천히 시작해도 돼"
 - 기쁨/평온: "좋은 아침! 오늘도 좋은 하루 보내자!"
    ↓
[오늘의 미션 카드 확인] (3개 제시)
    ↓
[하나 선택 후 앱 종료 or 미션 즉시 수행]
```

### 2.2 낮 플로우 (12:00~14:00)
```
[점심 알림 (선택): "베니가 안부를 묻고 있어요"]
    ↓
[선택: 감정 체크인 2회차]
    ↓
[정원 방문 → 오브젝트 확인]
    ↓
[미션 완료 체크 (오전 수행분)]
    ↓
[베니 대화 (베니 3단계 이상)]
    ↓
[앱 종료]
```

### 2.3 저녁/밤 플로우 (21:00~23:00)
```
[저녁 알림: "오늘 하루 어땠어? 베니가 기다려요"]
    ↓
[저녁 감정 체크인 (3회차)]
    ↓
[오늘의 미션 최종 완료 체크]
    ↓
[정원 꾸미기 (획득 오브젝트 배치)]
    ↓
[성장 앨범 → 오늘의 기록 확인]
    ↓
[베니 성장 확인 (경험치 변화)]
    ↓
[내일 미션 예고]
    ↓
[앱 종료]
```

## 3. 감정 체크인 여정

### 3.1 기쁨 여정
```
[기쁨 선택] → [베니 활짝 웃음 + 춤] → [정원: 햇살 & 꽃] 
→ [미션: "오늘 기쁜 일을 가까운 사람에게 공유해보세요"] 
→ [공유 완료] → [베니: "함께 기뻐해줄 사람이 있다는 게 얼마나 소중한지 알아?"]
```

### 3.2 평온 여정
```
[평온 선택] → [베니 온화한 미소] → [정원: 맑고 잔잔]
→ [미션: "지금 이 순간 내 주변 3가지 색깔을 찾아보세요 (마음챙김)"]
→ [완료] → [베니: "지금 이 순간에 머무르는 것, 충분해"]
```

### 3.3 피곤 여정
```
[피곤 선택] → [베니 눈 처짐, 하품] → [정원: 흐릿한 하늘]
→ [베니: "많이 지쳤구나. 오늘은 쉬어도 정말 괜찮아"]
→ [미션: "5분만 눈 감고 쉬어보세요" (쉬운 미션 제공)]
→ [완료] → [베니: "잘했어. 쉬는 것도 용기야"]
```

### 3.4 슬픔 여정
```
[슬픔 선택] → [베니 조용히 앉아 함께 있기] → [정원: 비 내림]
→ [베니: "그랬구나... 무슨 일이 있었어? (선택적)"]
→ [메모 입력 or 스킵]
→ [미션: "따뜻한 음료 한 잔 마셔보세요"]
→ [완료] → [베니: "베니는 항상 여기 있을게"]
→ [슬픔 강도 4~5 시: AI 위로 대화 모드 자동 시작]
```

### 3.5 불안 여정
```
[불안 선택] → [베니 몸 살짝 떨림] → [정원: 바람 부는 날]
→ [베니: "불안할 때는 숨을 한번 깊게 쉬어볼까?"]
→ [호흡 가이드 미니 기능 (4-7-8 호흡)]
→ [미션: "지금 내가 통제할 수 있는 것 하나를 적어보세요"]
→ [완료] → [베니: "넌 할 수 있어. 하나씩 해나가면 돼"]
→ [불안 강도 4~5 시: ACT 불안 대처 대화 모드 시작]
```

## 4. 베니 성장 여정

### 4.1 D1 — 첫 만남

- 첫 감정 체크인 완료
- 정원 기본 세팅 (씨앗 화분 1개)
- **베니**: "안녕! 나는 베니야. 우리 함께 피어나보자!"

### 4.2 D7 — 루틴 형성
- 7일 연속 출석 달성 → 베니 2단계로 성장 (꽃봉오리)
- 정원에 오브젝트 3~5개 배치됨
- 주간 감정 리포트 첫 제공 (기본)
- **알림**: "1주일 동안 함께해줘서 고마워! 베니가 많이 자랐어 🌸"

### 4.3 D30 — 패턴 인식
- 베니 3~4단계 (개화 ~ 열매)
- AI 대화 기능 완전 활성화
- 30일 감정 달력 완성
- 월간 AI 리포트 제공 (구독)
- **베니**: "한 달 동안 정말 많이 성장했어. 내가 느끼는 게 보여?"

### 4.4 D90 — 완전한 성장
- 베니 5단계 (나무)
- 성장 앨범 완성 (90일 여정 아카이빙)
- 특별 스페셜 씬 재생 (베니의 편지)
- **베니**: "우리가 함께 이만큼 피어났어. 처음 그날이 기억나?"

## 5. 핵심 사용자 시나리오

### 시나리오 1 — "번아웃 직전의 월요일 아침" (페르소나 A: 지수)

**상황**: 월요일 출근길, 극도의 피로감과 무기력함

**행동**:
1. 알림 확인 후 지하철에서 앱 실행
2. 감정: 피곤 / 강도 4 선택
3. 메모: "또 월요일이다. 아무것도 하기 싫어"
4. 베니가 조용히 옆에 앉아있는 애니메이션 재생
5. 베니 대사: "많이 지쳤구나. 오늘 아무것도 안 해도 괜찮아. 그냥 여기 있어"
6. 미션 제안: "오늘 딱 한 가지, 물 한 잔만 마셔보세요"
7. 출근 후 미션 완료 체크

**결과**: 작은 성취감 + "나를 봐주는 존재"에 대한 위안 → 다음 날 재접속

### 시나리오 2 — "갑작스러운 불안" (페르소나 B: 민준)

**상황**: 지도교수 피드백 후 논문 방향 전면 수정 통보, 패닉 상태

**행동**:
1. 강의실 화장실에서 앱 실행
2. 감정: 불안 / 강도 5 선택
3. 베니: 몸 떨림 + "괜찮아? 지금 많이 힘들어 보여"
4. AI 위로 대화 모드 자동 시작
5. 베니: "지금 가장 무서운 게 뭐야?" → 텍스트 입력
6. ACT 리프레이밍: "그 생각이 드는 건 자연스러워. 지금 네가 할 수 있는 가장 작은 것 하나는 뭘까?"
7. 호흡 가이드 3분 실행
8. 미션: "지금 당장 해결 못해도 되는 것 목록 1개 써보기"

**결과**: 패닉 감소, 현재 할 수 있는 것에 집중 → 앱에 대한 신뢰 형성

---

### 시나리오 3 — "30일 성장 달성" (페르소나 C: 수연)

**상황**: 아이가 낮잠 자는 30분, 베니 30일 성장 알림 수신

**행동**:
1. 알림: "베니가 30일 여행의 이정표에 도달했어요! 🍃"
2. 앱 실행 → 베니 열매 단계 성장 애니메이션
3. 30일 감정 달력 확인: 피곤이 가장 많았고 점차 평온이 늘어남
4. AI 월간 리포트: "이번 달 당신은 피곤한 날이 많았지만, 3주차부터 평온한 날이 늘었어요. 미션을 17개 완료했어요!"
5. 베니: "엄마로서, 한 사람으로서 정말 열심히 했어"
6. 특별 정원 오브젝트 "열매 나무" 획득

**결과**: 한 달간의 자기 돌봄이 가시적으로 확인됨 → 구독 전환 결심
