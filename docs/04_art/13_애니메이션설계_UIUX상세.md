## 2. 애니메이션 상태 머신 (Animator)

```
[Entry]
    ↓
[Idle_Sway] ← 기본 루프 상태
    ├── trigger: CheckinEmotion → [Checkin_[Emotion]] → Back to Idle
    ├── trigger: MissionComplete → [Mission_Complete] → Back to Idle
    ├── trigger: GrowStage → [Grow_Stage] → Back to Idle (높은 우선순위, 모든 상태에서 전환 가능)
    ├── trigger: TapBenny → [Tap_Reaction] → Back to Idle
    ├── bool: IsSleeping=true → [Sleep] ↔ Idle (시간 기반 자동 전환)
    └── trigger: DayGreeting → [Morning_Greeting] → Back to Idle

[Checkin_Joy] → 춤(Dance) 분기:
    - intensity >= 4: trigger: HighJoy → [Dance] → Back to Idle
    - intensity < 4: 직접 Idle 복귀

[Checkin_Sad/Tired] → 위로(Comfort) 분기:
    - intensity >= 4: trigger: DeepSad → [Comfort] → Back to Idle
```

### 트랜지션 설정
- **전환 시간**: 0.15s (빠른 감정 반응) ~ 0.3s (부드러운 상태 변화)
- **Has Exit Time**: 감정 반응 클립은 false (즉시 전환)
- **Interrupt Source**: Grow_Stage는 AnyState에서 전환 (최고 우선순위)

## 5. 인터랙션 플로우

| 인터랙션 | 동작 | 결과 |
|---------|------|------|
| **탭** | 감정 아이콘 탭 | 선택 상태 강조 + 햅틱 피드백 |
| **슬라이더 드래그** | 강도 슬라이더 | 실시간 베니 표정 미리보기 |
| **베니 탭** | 베니 직접 탭 | `ANIM_Tap_Reaction` + 랜덤 대사 |
| **스와이프업** | 정원 인벤토리 | 하단에서 인벤토리 서랍 슬라이드업 |
| **드래그앤드롭** | 오브젝트 배치 | 그리드 셀에 스냅, 배치 불가 시 진동 |
| **롱프레스** | 정원 오브젝트 | 이동/삭제 컨텍스트 메뉴 표시 |
| **스와이프좌우** | 미션 카드 | 카드 전환 |
| **핀치줌** | 정원 화면 | 확대/축소 (최소 0.7x ~ 최대 1.5x) |

## 6. 화면 전환 시나리오

| 출발 → 목적 | 전환 애니메이션 | 딜레이 |
|-----------|--------------|--------|
| 메인홈 → 정원 | 아래에서 위로 슬라이드업 | 0s |
| 메인홈 → 앨범 | 오른쪽에서 슬라이드인 | 0s |
| 어디서나 → 설정 | 오른쪽에서 슬라이드인 (half sheet) | 0s |
| 체크인 팝업 등장 | Fade + Scale (0.9→1.0) | 0s |
| 체크인 완료 | Fade out + 베니 반응 | 0.3s |
| 성장 달성 팝업 | 파티클 + Scale in (특별 연출) | 0.5s |
| 씬 전환 | 검은 Fade in/out (로딩 화면) | 로딩 완료까지 |

## 7. 접근성 고려사항

| 항목 | 기준 |
|------|------|
| **최소 폰트 크기** | 13px (Caption), 권장 16px+ |
| **컬러 대비** | 텍스트/배경 4.5:1 이상 (WCAG AA) |
| **터치 타겟** | 최소 44×44px |
| **색맹 대응** | 감정 표현 시 색상 + 아이콘 병용 (색만으로 구분 금지) |
| **다이나믹 폰트** | iOS/Android 시스템 폰트 크기 설정 반영 |
| **모션 감소** | 애니메이션 줄이기 설정 지원 (iOS Reduce Motion) |
