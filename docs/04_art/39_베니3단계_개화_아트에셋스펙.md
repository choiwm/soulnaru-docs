## 2. 산출물 목록 및 스펙


### 2-1. 2D 이미지 제작


  | 산출물 | 파일 형식 | 2D 해상도 수 | 해상도 단계 | 상태 |

  | 베니 3단계 메인 이미지 | PNG 스프라이트 + OBJ | ≤ 1,000 폴리 | 해상도 단계0: 1000 / 해상도 단계1: 500 / 해상도 단계2: 200 | ✅ 완료 |


  **이미지 제작 상세 규격:**

    - 단위: 미터법 (1unit = 1m)
    - 베니 전체 높이: 0.6m
    - 피벗 포인트: 발바닥 중앙 (0, 0, 0)
    - 정면 방향: +Z축
    - UV 맵: 단일 UV 채널, 0~1 범위, 겹침 없음
    - PNG 스프라이트 버전: PNG 스프라이트 2020 이상
    - 애니메이션: 별도 PNG 스프라이트 (T-pose 바인드포즈 포함)
    - 뼈대(Rig): Humanoid + 꼬리 체인 (3조인트) + 꽃잎 본 (4개) 추가



#### 3단계 전용 이미지 특징


  - 몸통: 꽃잎 4개 패턴 부조 (메시에 직접 이미지 제작)
  - 머리: 꽃 왕관 장식 (꽃잎 5개, 별도 오브젝트로 분리)
  - 꼬리: 끝부분에 꽃봉오리 장식 (단계 전환 애니메이션에 활용)
  - 눈: 2단계 대비 홍채 크기 10% 증가, 반짝임 파티클 파티클 본 포함


### 2-2. 이미지 레이어


  | 산출물 | 파일 형식 | 해상도 | 채널 | 상태 |

  | BaseColor (Diffuse) 맵 | PNG | 512×512px | RGB | ✅ 완료 |

  | Shadow 맵 | PNG | 512×512px | RGB (DirectX 기준) | ✅ 완료 |

  | Highlight 맵 (눈 반짝임) | PNG | 512×512px | RGB | ✅ 완료 |


  **이미지 레이어 색상 스펙 — 3단계 개화:**

    - 몸통 메인: `#A78BFA` (라벤더 바이올렛)
    - 꽃잎 장식: `#FCA5A5` (피치 핑크)
    - 꽃잎 내부/수술: `#FEF3C7` (연한 노랑)
    - 눈 홍채: `#7C3AED` (진한 바이올렛)
    - 눈 흰자: `#FFFFFF`
    - 코/입: `#F9A8D4` (연한 핑크)
    - 배경 그림자 영역: `#8B5CF6` × 50% 투명도
    - Highlight (눈 반짝): `#EDE9FE` (연한 라벤더, 발광)

  **스프라이트 제작 가이드:**

    - 캔버스: 256×256px, 투명 배경(PNG-24)
    - 베니 크기: 캔버스의 80% (약 205×205px), 하단 정렬

    - 강도 1→5: 표현 강도 점진적 증가. 5는 과장 표현 허용 (눈물 반짝, 꽃잎 떨림 등)

    - 아웃라인: 2px, `#5B21B6`
