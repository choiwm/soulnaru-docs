## 주요 개선 사항

### UI (CX 반영)
- **CX-06**: Hold-in 상태 #5B21B6 다크퍼플 실제 이미지에 반영
- **CX-07**: 접근성 — 세로 진행 바, 대형 타이머, 이모지+텍스트 동시 표시
- **CX-01**: 피곤 감정 제거 → 5감정 체계 확정

### 스프라이트 (CX-11 반영)
- 3단계 꽃잎 5개 장식 + #FCA5A5 피치핑크 뺨
- 4단계 열매 4개 장식 + #FCD34D 골든옐로 뺨
- 5단계 나뭇잎 6개 장식 + #4ADE80 라임그린 뺨
- 감정별 강도 5단계 세밀 표정 (눈물·땀방울·치아)

### 텍스처
- Albedo: 그라디언트 + 액센트 패턴
- Normal: 구형 굴곡 + 랜덤 범프 디테일
- Emission: 눈 + 단계별 장식 발광
- 계절 Albedo: 봄 벚꽃, 여름 잎, 가을 단풍, 겨울 눈송이
