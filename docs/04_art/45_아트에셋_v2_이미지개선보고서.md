## 4. 이미지 레이어 v2 개선 내역



    v1 (이전)

      - 단순 단색 배경
      - Shadow 맵 평면 (128,128,255)
      - Highlight 발광 영역 단순
      - 계절 이미지 레이어 패턴 없음



    v2 (현재)

      - BaseColor: 그라디언트 + 단계별 액센트 패턴 (CX-11 컬러 적용)
      - Shadow: 구형 굴곡 + 랜덤 범프 디테일
      - Highlight: 눈 발광 + 단계별 장식 발광 (열매/나뭇잎)
      - 계절 BaseColor: Spring 벚꽃, Summer 잎, Autumn 단풍, Winter 눈송이
