## 📋 최종 납품 현황



### 🎁 납품 파일 구성


        **3D 모델:** 15개 FBX 파일 (총 11.5MB)
        - **텍스처:** 45개 PNG 파일 (512×512px, 텍스처 + 노멀맵) (총 28MB)
        - **Prefab:** 15개 Unity Prefab 파일 (Collider, Material 포함)
        - **파티클 시스템:** 3개 파티클 Prefab (무지개, 별빛, 반딧불)
        - **Material:** 18개 커스텀 Material (감정별 색상 변형 포함)
        - **스크립트:** 2개 연동 스크립트 (감정별 색상 변경, 시즌 활성화)
        - **문서:** 모델링 가이드 + 사용 설명서 (4페이지)



        **📦 납품 폴더 구조:**

```
├── Textures/

    └── ArtTeam_ModelDelivery_Guide.pdf
```

        - 모든 모델 폴리곤 수 1000~2000 범위 내 (성능 최적화)

        - 모든 Prefab에 Collider 설정 (드래그&드롭 상호작용)
