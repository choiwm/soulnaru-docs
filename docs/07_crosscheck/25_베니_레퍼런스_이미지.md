## 📐 단계별 설계 기준

      1단계 (새싹 베니):** 작은 초록 새싹 형태로 순수하고 귀여운 느낌. 화분에서 나온 작은 크기의 캐릭터.
      **2단계 (꽃봉오리 베니):** 새싹에서 한 단계 성장하여 보라색 꽃봉오리가 생김. 표정이 더 다양해지고 생기있음.
      **3단계 이상:** 꽃이 활짝 피고, 팔(잎)이 생기며, 최종 5단계에는 작은 나무 형태로 완성됨.


## 🎨 컬러 팔레트 정의

      **1단계:** 주 컬러  #86EFAC (연초록) / 강조  #4ADE80 (진초록)
      **2단계:** 주 컬러  #6EE7B7 (민트) / 강조  #A78BFA (보라 꽃)
      **3단계:** 주 컬러  #A78BFA (보라 꽃) / 강조  #FCA5A5 (핑크 꽃심)
      **4단계:**  +  / 열매  #FCD34D (노란)
      **5단계:** 전 단계 종합 + 계절별 변화


## ✨ 감정 표현 가이드

      **기쁨:** 눈이 초승달 모양, 입이 활짝 웃음, 볼에 하트, 통통 튀는 느낌
      **평온:** 눈이 반쯤 감김, 온화한 미소, 부드러운 흔들림
      **피곤:** 눈이 처진 표정, 하품, 몸이 구부러짐
      **슬픔:** 눈썹이 처짐, 입꼬리 하향, 눈물 표현
      **불안:** 눈이 크게 흔들림, 몸 좌우 떨림, 불편한 자세


## 💾 아트팀 제작 체크리스트


        ✅ 1단계 (새싹): SVG 벡터 + PNG 렌더링 (256×256px, 512×512px)
        - ✅ 2단계 (꽃봉오리): SVG 벡터 + PNG 렌더링
        - ⏳ 3단계 (개화): 꽃 활짝 핀 상태 + 팔(잎) 추가
        - ⏳ 4단계 (열매): 열매 2~3개 + 풍성한 잎
        - ⏳ 5단계 (나무): 작은 나무 형태 완성
        - ⏳ 각 단계별 감정 스프라이트 (5감정 × 5강도 = 25개)
        - ⏳ Animator Override Controller 클립 매핑
