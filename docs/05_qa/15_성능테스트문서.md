## 3. FPS 측정 방법 및 기준

### 측정 방법
1. **Unity Profiler**: 에디터 내 각 씬별 FPS 측정
2. **GameBench**: 실기기 30분 세션 FPS 모니터링
3. **기기 내 개발자 모드**: GPU 렌더링 프로파일 활성화

### 측정 시나리오
| 시나리오 | 측정 항목 |
|---------|---------|
| 메인 홈 대기 | 베니 Idle 애니메이션 중 FPS |
| 감정 체크인 완료 | 베니 감정 애니메이션 재생 중 FPS |
| 정원 오브젝트 20개 배치 | 최대 오브젝트 상태 FPS |
| 날씨 파티클 활성화 | 비/눈 파티클 풀 활성화 FPS |
| 씬 전환 직후 | 씬 로딩 완료 직후 FPS 안정화 시간 |

### Pass 기준
- 저사양 기기: 평균 30FPS, 최저 25FPS
- 중/고사양 기기: 평균 60FPS, 최저 50FPS

## 4. 메모리 사용량 측정

### 씬별 메모리 목표
| 씬 | 목표 메모리 | 주요 부하 요소 |
|----|-----------|--------------|
| Boot | 50MB 이하 | 초기화, 에셋 로딩 |
| MainHome | 120MB 이하 | 베니 모델, UI, 파티클 |
| Garden | 150MB 이하 | 오브젝트 다수, 날씨 파티클 |
| Album | 100MB 이하 | 이미지 다수 (달력) |

### 측정 도구
- **Android**: Android Studio Memory Profiler
- **iOS**: Xcode Instruments (Leaks + Allocations)

### 메모리 누수 체크
- 씬 전환 후 이전 씬 메모리 해제 확인
- 30분 장시간 세션 후 메모리 증가량 < 20MB

## 5. 로딩 시간 측정

| 측정 항목 | 방법 | 목표 |
|---------|------|------|
| 앱 콜드 스타트 | 아이콘 탭 → 메인 홈 표시까지 | 3초 이내 |
| 메인홈 → 정원 | 탭 탭 → 정원 씬 표시까지 | 2초 이내 |
| 메인홈 → 앨범 | 탭 탭 → 앨범 씬 표시까지 | 2초 이내 |
| 감정 체크인 → 베니 반응 | 완료 버튼 → 애니메이션 시작까지 | 0.5초 이내 |
