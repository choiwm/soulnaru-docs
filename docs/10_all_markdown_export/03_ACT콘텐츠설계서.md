## 1. ACT 6가지 핵심 프로세스의 게임화

수용전념치료(ACT: Acceptance and Commitment Therapy)의 6가지 핵심 프로세스를 게임 메카닉으로 전환한다.

| ACT 프로세스 | 게임 메카닉 | 베니 역할 |
| --- | --- | --- |
| 1. 수용 (Acceptance) | 감정 체크인: 판단 없이 감정 선택 | "그 감정이 맞아. 괜찮아" 반응 |
| 2. 탈융합 (Defusion) | "내 생각 vs 사실" 미션 카드 | "그 생각이 전부가 아니야" 대화 |
| 3. 현재 순간 접촉 (Present Moment) | 마음챙김 미션 (3가지 색 찾기 등) | 지금 이 순간 집중 유도 대화 |
| 4. 맥락으로서의 자기 (Self-as-Context) | 30일/90일 앨범 회고 | "한 달 전 너와 지금 너를 비교해봐" |
| 5. 가치 (Values) | 가치 탐색 주간 미션 | "네가 정말 중요하게 여기는 게 뭐야?" 심층 대화 |
| 6. 전념 행동 (Committed Action) | 루틴 미션 완료 체크 | 행동 완료 시 칭찬 + 연속 강화 |

## 2. 일일 미션 시나리오 7일치

### Day 1 — 나를 만나는 날 (수용)

| 항목 | 내용 |
| --- | --- |
| 미션명 | "오늘의 나에게 인사하기" |
| 목표 | 자기 감정 수용 시작 |
| 행동 | 감정 체크인 후 거울 앞에서 "오늘 나는 [감정]해. 괜찮아"라고 말하기 |
| 베니 반응 | "맞아, 그 감정 그대로도 괜찮아. 베니는 다 알아" |
| 보상 | 씨앗 +20, 배지: "첫 만남" |

### Day 2 — 몸을 깨우는 날 (전념 행동)

| 항목 | 내용 |
| --- | --- |
| 미션명 | "몸에게 선물 한 가지" |
| 목표 | 신체 케어 습관 시작 |
| 행동 | 물 2잔 마시기 + 팔 스트레칭 2분 |
| 베니 반응 | "몸을 돌보는 것도 나를 사랑하는 방법이야" |
| 보상 | 씨앗 +15, 정원 오브젝트: 작은 화분 |

### Day 3 — 생각을 바라보는 날 (탈융합)

| 항목 | 내용 |
| --- | --- |
| 미션명 | "내 생각에 이름 붙이기" |
| 목표 | 생각과 자신을 분리하는 연습 |
| 행동 | 요즘 자주 드는 부정적 생각 1가지를 적고, 그 앞에 "나는 지금 [생각]이라는 생각을 하고 있어" 라고 다시 쓰기 |
| 베니 반응 | "그 생각이 너 전체는 아니야. 그냥 지나가는 손님 같은 거야" |
| 보상 | 씨앗 +20, 베니 대화 해금 |

### Day 4 — 지금 여기에 있는 날 (현재 순간)

| 항목 | 내용 |
| --- | --- |
| 미션명 | "지금 이 순간 탐험" |
| 목표 | 마음챙김 연습 |
| 행동 | 5분 동안: 보이는 것 3가지, 들리는 소리 2가지, 느껴지는 감각 1가지 적기 |
| 베니 반응 | "지금 이 순간에 있어줘서 고마워. 베니도 함께 있었어" |
| 보상 | 씨앗 +20, 정원 날씨 효과: 햇살 추가 |

### Day 5 — 감사를 찾는 날 (가치)

| 항목 | 내용 |
| --- | --- |
| 미션명 | "오늘의 작은 선물" |
| 목표 | 긍정 경험 인식 및 가치 연결 |
| 행동 | 오늘 감사한 것 3가지 적기 (아무리 작아도 됨) |
| 베니 반응 | "이런 것들이 네 삶에 있다는 거, 꽤 특별하지 않아?" |
| 보상 | 씨앗 +25, 배지: "감사의 씨앗" |

### Day 6 — 연결하는 날 (전념 행동 + 가치)

| 항목 | 내용 |
| --- | --- |
| 미션명 | "소중한 사람에게 안부 전하기" |
| 목표 | 관계 가치 실천 |
| 행동 | 오랫동안 연락하지 못한 사람 1명에게 짧은 메시지 보내기 |
| 베니 반응 | "작은 연결이 세상을 따뜻하게 만들어. 베니도 기분이 좋아졌어" |
| 보상 | 씨앗 +30, 정원 오브젝트: 작은 벤치 |

### Day 7 — 한 주를 돌아보는 날 (맥락으로서의 자기)

| 항목 | 내용 |
| --- | --- |
| 미션명 | "이번 주 나에게 편지 쓰기" |
| 목표 | 1주일 성장 회고 |
| 행동 | 이번 주 나에게 한 줄 편지: "이번 주 나는 [ ]했고, 수고했어" |
| 베니 반응 | "와, 이번 주 정말 많이 했잖아. 베니는 다 봤어" |
| 보상 | 씨앗 +50, 7일 연속 배지, 베니 성장 단계 확인 |

## 3. 감정별 대응 미션 매핑

| 감정 | 강도 1~2 | 강도 3 | 강도 4~5 |
| --- | --- | --- | --- |
| 기쁨 | 감사 일기 쓰기 | 기쁨 공유 미션 | 특별 기쁨 이벤트 알림 |
| 평온 | 마음챙김 5분 | 가치 탐색 미션 | 심층 성찰 대화 |
| 피곤 | 물 한 잔 마시기 | 5분 눈 감고 쉬기 | 오늘 미션 1개로 축소 + AI 위로 |
| 슬픔 | 감정 일기 쓰기 | 따뜻한 음료 미션 | AI 위로 대화 강제 활성화 |
| 불안 | 호흡 가이드 3분 | 통제 가능한 것 리스트 | 위기 대응 + 심리상담 리소스 안내 |

## 4. 심리적 효과 측정 방법

### 4.1 인앱 지표

| 지표 | 측정 방법 | 측정 주기 |
| --- | --- | --- |
| 감정 다양성 지수 | 5감정 중 몇 가지를 기록했는지 | 주간 |
| 부정 감정 강도 평균 | 피곤+슬픔+불안 강도 평균 | 주간/월간 |
| 미션 완료율 | 완료 미션 수 / 제공 미션 수 | 주간 |
| 연속 출석일 | 스트릭 데이터 | 실시간 |
| AI 대화 참여율 | 대화 실행 / 유도 횟수 | 주간 |

### 4.2 유저 설문 (인앱 팝업)

- D7 : "이번 주 앱을 사용하면서 감정 변화가 있었나요?" (1~5점)

- D30 : "베니와 함께 이후 일상에서 변화가 있었나요?" (주관식 + 점수)

- D90 : 심리적 웰빙 척도 간이 측정 (WHO-5 5문항)
