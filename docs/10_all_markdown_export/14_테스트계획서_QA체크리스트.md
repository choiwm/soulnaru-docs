## 3. 통합 테스트 시나리오

### 시나리오 1 — 전체 일일 루틴 흐름

`1. 앱 실행 → 메인 홈 로딩
2. 감정 체크인 (슬픔/강도3) 완료
3. 베니 반응 확인 (위로 애니메이션 재생 여부)
4. 정원 날씨 변화 확인 (비로 변경)
5. 미션 카드 확인 (슬픔에 맞는 쉬운 미션 3개)
6. 미션 1개 완료 → 보상 지급 확인
7. 정원 이동 → 오브젝트 배치
8. 앨범 이동 → 오늘 감정 기록 반영 확인
9. 앱 종료 → 재실행 → 데이터 유지 확인` 

### 시나리오 2 — 오프라인 상황

`1. 기기 비행기 모드 ON
2. 감정 체크인 수행 → 로컬 저장 확인
3. 미션 완료 → 로컬 저장 확인
4. 비행기 모드 OFF
5. 서버 동기화 자동 실행 확인
6. 서버 데이터와 로컬 데이터 일치 검증` 

### 시나리오 3 — 구독 결제 플로우

`1. 설정 → 프리미엄 구독 탭
2. 월 3,900원 상품 선택
3. Google Play/App Store 결제 창 표시
4. 결제 완료 → 서버 검증
5. 구독 혜택 즉시 활성화 확인 (AI 리포트 등)
6. 구독 취소 후 기능 비활성화 확인`

## 5. 유저 테스트 (FGT)

### 5.1 알파 테스트 (5월)

- 대상 : 내부 팀원 + 지인 10명 (타겟 연령대)

- 기간 : 10일 연속 사용 (D7 리텐션 + D10 마무리 설문 확보)

- 측정 : D1/D7/D10 리텐션, 완료된 미션 수, 감정 체크인 빈도

### 5.2 베타 FGT (6월)

- 대상 : 외부 50명 (20~35세, 감정 케어 앱 경험자 우선)

- 기간 : 2주

- 측정 지표 :

- 첫 체크인까지 시간 (온보딩 이탈율)

- D1/D7 리텐션

- 세션 길이 평균

- 구독 의향 설문 (1~5점)

#### 5.2.1 FGT 참가자 사전 스크리닝 설문

모집 단계에서 아래 3문항으로 적합 참가자를 선별한다. **1번 또는 2번 해당자 우선 선발**.

| # | 질문 | 선발 기준 |
| --- | --- | --- |
| Q1 | 최근 3개월 내 명상 / 감정 기록 / 정신건강 앱(예: 마보, Calm, 마인드카페 등)을 사용한 경험이 있나요? | 있음 → 우선 선발 |
| Q2 | 최근 6개월 내 스트레스 또는 번아웃을 경험하고 있거나 경험한 적이 있나요? | 있음 → 우선 선발 |
| Q3 | 모바일 게임을 주 3회 이상 이용하나요? | 있음 → 보조 선발 |

⚠️ Q1 또는 Q2 해당자 40명 + Q3만 해당자 10명 비율로 50명 구성 권장 
⚠️ 현재 정신건강 위기 상황(치료 중 포함)인 분은 참가 배제 (사전 안내 필수) 

- FGT 설문 항목 (사용 후 측정):

- "베니가 내 감정을 이해해주는 것 같았나요?" (1~5점)

- "이 앱을 1주일 이상 계속 쓰고 싶은가요?" (1~5점)

- "이 앱을 친구나 지인에게 추천할 의향이 있나요?" (NPS: 0~10점)

- "가장 좋았던 기능은?" (주관식)

- "개선이 필요한 점은?" (주관식)
