# 소울나루 전체 생성 문서 Markdown Export

- 생성일: 2026-05-01
- 원본 웹 포털: https://lrndxihi.gensparkclaw.com/benny/
- 전체 Markdown 다운로드: https://lrndxihi.gensparkclaw.com/benny/soulnaru_all_markdown_docs_20260501.zip
- HTML + Markdown 전체 다운로드: https://lrndxihi.gensparkclaw.com/benny/soulnaru_all_docs_html_and_markdown_20260501.zip

이 폴더는 `/var/www/benny/`의 생성 HTML 문서를 Markdown으로 일괄 변환한 export입니다.
이미지/에셋 참조를 위해 `assets/` 폴더도 함께 포함합니다.

## 사용 방법

1. GitHub에서 이 폴더를 확인하거나 ZIP을 다운로드합니다.
2. 개별 문서는 같은 폴더의 `.md` 파일을 열어 확인합니다.
3. 이미지가 포함된 문서는 `assets/` 경로를 함께 유지해야 합니다.
