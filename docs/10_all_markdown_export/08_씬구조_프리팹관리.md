## 3. 씬 전환 트리거 및 데이터 전달

### 3.1 씬 전환 트리거

| 출발 씬 | 목적 씬 | 트리거 |
| --- | --- | --- |
| Boot | MainHome | 로그인 완료 + 데이터 로딩 완료 |
| Boot | Onboarding | 첫 실행 감지 |
| Onboarding | MainHome | 튜토리얼 5단계 완료 |
| MainHome | Garden | 하단 탭 '정원' 탭 |
| MainHome | Album | 하단 탭 '앨범' 탭 |
| Any | MainHome | 하단 탭 '홈' 탭 |

### 3.2 씬 간 데이터 전달 방식

`// SceneTransitionData (DontDestroyOnLoad 싱글톤)
public class SceneTransitionData : MonoBehaviour
{
public static SceneTransitionData Instance;
public EmotionType LastCheckedEmotion;
public int LastIntensity;
public bool IsFirstLaunch;
}` 

- 주요 게임 상태는 GameManager 싱글톤에 보관

- 씬 전환 파라미터는 SceneTransitionData 로 전달

- 영구 데이터는 DataSyncManager 를 통해 서버/로컬 동기화

## 5. UI UI 프리팹 버전 관리 규칙

- UI UI 프리팹 수정 시 반드시 UI UI UI 프리팹 Mode 에서 수정 (씬에서 직접 수정 금지)

- 공유 UI UI 프리팹 수정 전 팀 Slack 채널에 사전 공지

- UI UI 프리팹 파괴 및 새로 생성 금지 — Override 방식 활용

- 주요 UI UI 프리팹 변경 시 변경 이력 주석 추가:

`// [2026-04-15] BennyController: 감정 애니메이션 트랜지션 속도 0.3s→0.5s 변경 (홍길동)`
