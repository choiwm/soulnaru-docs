## 4. StreakRewardManager 구현 가이드

**로직:** 3일·7일·14일·30일 스트릭 달성 시 자동으로 오브젝트 1개씩 부여. 중복 방지.

// StreakRewardManager.cs
public class StreakRewardManager : MonoBehaviour {
private static readonly Dictionary STREAK_OBJECT_MAP = new() {
{ 3, 1 }, // 3일 → 1번 새싹 씨앗
{ 7, 2 }, // 7일 → 2번 흰 꽃
{ 14, 3 }, // 14일 → 3번 보라 꽃나무
{ 30, 6 } // 30일 → 6번 파란 수국
};

public async Task CheckAndGrantStreakRewards(int currentStreak, string userId) {
foreach (var (streakDay, objectId) in STREAK_OBJECT_MAP) {
if (currentStreak == streakDay) {
// 중복 확인
bool alreadyOwned = await GardenManager.HasObject(userId, objectId);
if (!alreadyOwned) {
// 오브젝트 부여
await GardenManager.AddObject(userId, objectId, source: "streak_reward");
// Firestore 기록
await FirestoreManager.LogStreakReward(userId, streakDay, objectId);
// 베니 특별 애니메이션 재생
BennyAnimator.PlayStreakAnnounce(streakDay);
}
}
}
}
} 

## 5. GardenManager 통합 (기존 로직 변경 없음)

기존 GardenManager의 "미션 완료 시 오브젝트 부여" 로직은 **1번(새싹)을 제외한 일반 오브젝트**만 담당:

// GardenManager.cs — 기존 로직 유지
public async Task OnMissionComplete(Mission mission, string userId) {
// 스트릭 체크 (StreakRewardManager 위임)
await StreakRewardManager.CheckAndGrantStreakRewards(
currentStreak: _streakCounter.GetCurrentStreak(),
userId: userId
);

// 일반 미션 보상 (스트릭 보상 제외)
switch (mission.Day) {
case 4: await AddObject(userId, 4); // 4번 노란 해바라기
break;
// ... 기타 일반 보상
}
}
