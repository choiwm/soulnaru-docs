## 1️⃣ 기획팀 문서 리뷰 (01~03)

### ✅ 잘된 점
- GDD의 베니 5단계 성장 조건이 구체적 수치로 명시됨 (감정 체크인 횟수, 미션 완료 수)
- 유저 스토리 맵 페르소나 3종이 타겟 세그먼트를 잘 커버 (직장인/대학원생/육아맘)
- ACT 7일치 미션이 각 프로세스와 명확히 연결됨

### 🔴 Critical 이슈

**[GDD] Unity 버전 불일치**
- GDD 1.1: "Unity (LTS 최신 버전)" → 모호한 표현
- 개발팀 문서(07): "Unity 2022.3 LTS" → 구체적 명시
- **수정**: GDD에도 "Unity 2022.3 LTS"로 통일 필요

**[GDD] 등급 불일치**
- GDD: "전체이용가" → 감정 위기 대응 기능(자살예방 리소스 안내) 포함 시 재검토 필요
- 배포 체크리스트(18): "18세 이상 권장" 표시 명시
- **수정**: 등급을 "12세 이용가 또는 15세 이용가" 검토 필요, GDD와 18번 문서 일치시킬 것

### 🟡 Important 이슈

**[GDD] 성장 조건 난이도 검토 필요**
- 2단계: 체크인 20회 + 미션 15개 (약 5~7일 → 적절)
- 3단계: 체크인 50회 + 미션 40개 (약 17~20일 → 적절)
- 4단계: **연속 30일 출석 + 미션 80개 (동시 충족 어려움)**
  - 30일 연속 출석 + 미션 80개는 하루 평균 2.7개 미션 = 빡빡한 조건
  - **제안**: "30일 누적 출석(연속 불필요) + 미션 80개"로 완화 검토

**[GDD] 온보딩 상세 없음**
- 6.1 화면 목록에 "온보딩 5단계" 언급되나 각 단계 내용 미기술
- UI/UX 문서(13)와도 온보딩 내용 미연결
- **제안**: GDD 섹션 8로 "온보딩 플로우" 추가 필요

**[유저스토리맵] D30 전후 구독 유도 시점 불명확**
- 페르소나 C 시나리오에서 "30일 성장 달성 후 구독 결심" 묘사됨
- 그러나 AI 리포트(구독 전용)가 30일 이후 처음 제공된다면 → D30 전에 구독 유도 장치 없음
- **제안**: D7 주간 리포트 맛보기 제공 → 구독 유도 퍼널 추가

### 🟢 Minor 이슈
- GDD 2.1 코어 루프에서 "AI 베니 대화" 스텝이 누락됨 (대화는 3단계 이후 활성화임을 명시)

## 6️⃣ 비즈니스팀 문서 리뷰 (16~18)

### ✅ 잘된 점
- 17번 수익 예측 계산이 MAU × 전환율 × 단가 구조로 명확
- 리스크 관리(16번) 8가지 리스크에 발생 가능성 + 영향도 + 대응 전략 3단 구조 우수
- 18번 앱스토어 체크리스트가 Google Play / App Store 각각 분리됨

### 🔴 Critical 이슈

**[예산 17] 지원금 비율 계산 오류**
- 지원금 5,000만원 = 90% → 자부담 = 10% → 총예산 = 55,555,556원 (표기됨)
- 하지만 일반적으로 경기게임제작지원은 **지원금이 총예산의 90%** → 자부담은 총예산의 10%
- 즉 총예산 = 5,000만원 ÷ 0.9 = 55,555,556원 → 자부담 = 5,555,556원 ✅ (현재 계산은 맞음)
- BUT 지원금 공고 원문에서 "총 사업비의 10% 자부담" 또는 "지원금 5,000만원 별도 자부담 최소 금액" 중 어느 방식인지 확인 후 명확히 표기 필요
- **수정**: 공고 원문 기준 명확히 명시
