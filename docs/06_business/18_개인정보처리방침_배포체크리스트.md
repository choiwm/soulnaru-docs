## 1. 개인정보처리방침

시행일: 2026년 7월 1일

### 1.1 수집 항목

| 구분 | 수집 항목 | 필수 여부 |
|------|---------|---------|
| **계정 정보** | 이메일, 닉네임 | 필수 |
| **감정 데이터** | 감정 종류(5종), 강도(1~5), 선택적 메모 | 필수 (서비스 핵심) |
| **이용 패턴** | 앱 실행 시간, 미션 완료 기록, 정원 배치 정보 | 필수 |
| **결제 정보** | 구독/패스 구매 내역 (카드 번호 미수집 — 스토어 처리) | 선택 |
| **기기 정보** | 기기 모델, OS 버전, 앱 버전 (서비스 최적화) | 필수 |
| **알림 토큰** | FCM 푸시 토큰 | 선택 |

### 1.2 수집 목적 및 이용 범위

**법적 근거**: 「개인정보 보호법」 제15조 제1항 제1호에 따라 정보주체의 동의를 받아 수집합니다. 감정 데이터의 경우 「개인정보 보호법」 제23조(민감정보 처리 제한)에 따라 별도 동의를 받아 수집합니다.

| 목적 | 이용 항목 |
|------|---------|
| 서비스 제공 | 감정 반응, 베니 성장, 정원 관리 |
| AI 맞춤 추천 | 감정 데이터 → 맞춤 미션/대화 생성 |
| 통계 분석 | 익명화된 감정 패턴 → 서비스 개선 |
| 결제 처리 | 구독 상태 관리 (스토어 검증) |
| 고객 지원 | 문의 처리 시 계정 정보 확인 |

### 1.3 보관 기간 및 파기 방법

| 데이터 | 보관 기간 | 파기 방법 |
|--------|---------|---------|
| 계정 정보 | 탈퇴 후 30일 | 자동 삭제 |
| 감정 체크인 기록 | 탈퇴 후 30일 | 즉시 삭제 |
| 감정 메모 텍스트 | 유저 설정 (기본 1년) | 유저 직접 삭제 또는 자동 만료 |
| 이용 패턴 데이터 | 탈퇴 후 30일 | 자동 삭제 |
| 익명화 통계 데이터 | 최대 3년 | 개인 식별 불가 형태로 보관 |
| 결제 내역 | 관련 법령에 따라 5년 | 법령 보관 후 삭제 |

### 1.4 감정 데이터 특별 처리 원칙

감정 데이터는 민감한 개인정보에 해당할 수 있으므로 아래 원칙을 준수한다:

1. **최소 수집**: 서비스 제공에 필요한 최소한의 감정 정보만 수집
2. **익명화 우선**: AI 학습 및 통계 분석 시 개인 식별 불가 형태로 처리
3. **분리 보관**: 감정 데이터는 계정 정보와 분리 저장
4. **목적 외 사용 금지**: 마케팅, 제3자 판매에 감정 데이터 절대 사용 금지
5. **암호화**: 서버 저장 시 AES-256 암호화 적용
6. **메모 보호**: 텍스트 메모는 서버 저장 전 PII 마스킹 처리

### 1.5 제3자 제공 정책

| 제3자 | 제공 항목 | 제공 목적 | 보관 기간 |
|------|---------|---------|---------|
| **NC AI (VARCO)** | 에셋 생성 요청 데이터 (개인 식별 정보 미포함) | AI 에셋 생성 서비스 | 처리 후 즉시 삭제 |
| **Firebase (Google)** | 앱 이용 로그 (익명화) | 분석 및 알림 서비스 | Firebase 정책 준수 |
| **Apple/Google** | 결제 검증 데이터 | 구독 검증 | 각 스토어 정책 준수 |

**감정 데이터는 어떠한 제3자에게도 제공하지 않습니다.**

### 1.6 이용자 권리

| 권리 | 방법 |
|------|------|
